Infográfico: Inequações

Introdução

Inequações são expressões matemáticas que, diferentemente das equações (que usam o sinal de igualdade =), utilizam sinais de desigualdade para comparar grandezas. Elas são fundamentais para modelar situações onde existe uma relação de "maior que", "menor que", "maior ou igual a" ou "menor ou igual a".

Sinais de Desigualdade:

  • > : Maior que
  • < : Menor que
  • ≥ : Maior ou igual a (≥)
  • ≤ : Menor ou igual a (≤)

Resolver uma inequação significa encontrar o conjunto de todos os valores (conjunto solução) que tornam a desigualdade verdadeira.

Inequações do 1º Grau

São inequações que podem ser reduzidas à forma:

  • ax + b > 0
  • ax + b < 0
  • ax + b ≥ 0
  • ax + b ≤ 0

Onde a e b são números reais (ℝ) e a ≠ 0.

📝 Como Resolver:

  1. Isolar a incógnita (x): Utilize as mesmas técnicas de resolução de equações do 1º grau (operações inversas).
  2. Atenção ao sinal de a:
    • Se você multiplicar ou dividir ambos os lados da inequação por um número negativo, o sinal da desigualdade deve ser invertido.

Exemplo:

Resolver 2x - 6 > 0

  1. Adicionar 6 aos dois lados:
    2x - 6 + 6 > 0 + 6
    2x > 6
  2. Dividir os dois lados por 2 (número positivo, não inverte o sinal):
    2x / 2 > 6 / 2
    x > 3
Conjunto Solução (S): S = {x ∈ ℝ | x > 3}

📏 Representação Gráfica (Reta Numérica):

Para x > 3:

(----------> <------|----------|------> 3 x (Intervalo aberto em 3)

Para x ≤ -1:

<----------] <------|----------|------> x -1 (Intervalo fechado em -1)

Inequações do 2º Grau

São inequações que podem ser reduzidas à forma:

  • ax2 + bx + c > 0
  • ax2 + bx + c < 0
  • ax2 + bx + c ≥ 0
  • ax2 + bx + c ≤ 0

Onde a, b e c são números reais (ℝ) e a ≠ 0.

📝 Como Resolver:

  1. Encontrar as raízes da equação quadrática: Resolva ax2 + bx + c = 0 para encontrar as raízes x1 e x2 (usando a fórmula de Bhaskara: x = (-b ± √Δ) / (2a), onde Δ = b2 - 4ac).
  2. Analisar a concavidade da parábola:
    • Se a > 0: A parábola tem concavidade para cima ().
    • Se a < 0: A parábola tem concavidade para baixo ().
  3. Esboçar o gráfico da função quadrática: Marque as raízes no eixo x e desenhe a parábola de acordo com a concavidade.
  4. Identificar os intervalos: Analise no esboço do gráfico os intervalos onde a função é positiva (> 0), negativa (< 0), não negativa (≥ 0) ou não positiva (≤ 0), conforme a inequação original.

🔍 Casos de acordo com o Discriminante (Δ):

Δ > 0 (Duas raízes reais e distintas: x1 e x2)

a > 0 (Concavidade para cima):

  • ax2 + bx + c > 0 ⇒ x < x1 ou x > x2
  • ax2 + bx + c < 0 ⇒ x1 < x < x2

a < 0 (Concavidade para baixo):

  • ax2 + bx + c > 0 ⇒ x1 < x < x2
  • ax2 + bx + c < 0 ⇒ x < x1 ou x > x2

Exemplo Visual (a > 0):

+ | + / | \\ / | \\ ---(x₁)----(0)----(x₂)--- Eixo x | - (entre x₁ e x₂)
Δ = 0 (Uma raiz real dupla: x1 = x2 = x0)

a > 0 (Concavidade para cima):

  • ax2 + bx + c > 0 ⇒ x ≠ x0
  • ax2 + bx + c < 0 ⇒ Não existe solução (∅)
  • ax2 + bx + c ≥ 0 ⇒ x ∈ ℝ (Todos os reais)
  • ax2 + bx + c ≤ 0 ⇒ x = x0

a < 0 (Concavidade para baixo):

  • ax2 + bx + c > 0 ⇒ Não existe solução (∅)
  • ax2 + bx + c < 0 ⇒ x ≠ x0
  • ax2 + bx + c ≥ 0 ⇒ x = x0
  • ax2 + bx + c ≤ 0 ⇒ x ∈ ℝ (Todos os reais)

Exemplo Visual (a > 0):

+ | + / | \\ / | \\ ---------(x₀)--------- Eixo x (parábola toca o eixo)
Δ < 0 (Nenhuma raiz real)

a > 0 (Concavidade para cima):

  • ax2 + bx + c > 0 ⇒ x ∈ ℝ (Todos os reais)
  • ax2 + bx + c < 0 ⇒ Não existe solução (∅)

a < 0 (Concavidade para baixo):

  • ax2 + bx + c > 0 ⇒ Não existe solução (∅)
  • ax2 + bx + c < 0 ⇒ x ∈ ℝ (Todos os reais)

Exemplo Visual (a > 0):

+ + + + + + + / \\ / \\ (parábola não toca o eixo x, ----------------------- Eixo x toda acima do eixo)

Exemplo:

Resolver x2 - 5x + 6 > 0

  1. Raízes de x2 - 5x + 6 = 0:
    Δ = (-5)2 - 4 · 1 · 6 = 25 - 24 = 1
    x = (5 ± √1) / (2 · 1)
    x1 = (5 - 1) / 2 = 2
    x2 = (5 + 1) / 2 = 3
  2. Concavidade: a = 1 (positivo), então a concavidade é para cima ().
  3. Esboço e Análise:
    + | + / | \\ / | \\ ---(2)----(0)----(3)--- Eixo x | - (entre 2 e 3)

    A inequação é > 0 (positiva). Pelo esboço, a função é positiva quando x < 2 ou x > 3.

Conjunto Solução (S): S = {x ∈ ℝ | x < 2 \text{ ou } x > 3}

Sistemas de Inequações

Um sistema de inequações é um conjunto de duas ou mais inequações que devem ser satisfeitas simultaneamente.

📝 Como Resolver:

  1. Resolva cada inequação do sistema separadamente.
  2. Encontre a interseção (∩) dos conjuntos solução de cada inequação. A interseção pode ser visualizada comparando as representações gráficas (retas numéricas) das soluções individuais.

Exemplo:

Resolver o sistema:

  1. 2x - 4 > 0
  2. -x + 3 ≥ 0

Resolvendo (1):

2x > 4
x > 2

S1:

(----------> <------|----------|------> 2

Resolvendo (2):

-x ≥ -3 (Multiplicando por -1, inverte o sinal)
x ≤ 3

S2:

<----------] <------|----------|------> 3

Interseção S1 ∩ S2:

S₁: (------------------> <---|----|----|----|----|---> 0 1 2 3 4 S₂: <------------------] <---|----|----|----|----|---> 0 1 2 3 4 S₁∩S₂: (----------] <---|----|----|----|----|---> 0 1 2 3 4
Conjunto Solução (S): S = {x ∈ ℝ | 2 < x ≤ 3}

Inequação Produto e Inequação Quociente

Inequação Produto:

Forma: f(x) · g(x) > 0 (ou <, ≥, ≤)

Inequação Quociente:

Forma: f(x) / g(x) > 0 (ou <, ≥, ≤)

Importante: Para inequação quociente, g(x) ≠ 0.

📝 Como Resolver (Estudo de Sinais / "Varal de Sinais"):

  1. Encontrar as raízes de cada função: Determine os valores de x para os quais f(x) = 0 e g(x) = 0.
  2. Estudar o sinal de cada função separadamente: Analise os intervalos onde f(x) e g(x) são positivas ou negativas.
  3. Construir a tabela de sinais (varal):
    • Coloque as raízes encontradas em ordem crescente.
    • Indique os sinais de f(x) e g(x) em cada intervalo.
    • Determine o sinal do produto f(x) · g(x) ou do quociente f(x) / g(x) em cada intervalo usando a regra de sinais.
  4. Identificar a solução: Selecione os intervalos que satisfazem a desigualdade original.
    • Atenção: Para inequações quociente, os valores de x que tornam o denominador g(x) igual a zero NÃO fazem parte da solução.

Exemplo (Inequação Produto):

Resolver (x - 1)(x + 2) ≥ 0

  1. Raízes:
    • x - 1 = 0 ⇒ x = 1
    • x + 2 = 0 ⇒ x = -2
  2. Estudo de sinais:
    • f(x) = x - 1: (função do 1º grau, a=1 > 0, crescente)
      • x - 1 > 0 ⇒ x > 1 (positiva)
      • x - 1 < 0 ⇒ x < 1 (negativa)
    • g(x) = x + 2: (função do 1º grau, a=1 > 0, crescente)
      • x + 2 > 0 ⇒ x > -2 (positiva)
      • x + 2 < 0 ⇒ x < -2 (negativa)
  3. Tabela de Sinais:
    Intervalo (-∞, -2) -2 (-2, 1) 1 (1, +∞)
    f(x) = x-1 - - - 0 +
    g(x) = x+2 - 0 + + +
    f(x)g(x) + 0 - 0 +

    (Regra: (-) · (-) = (+); (-) · (+) = (-); (+) · (+) = (+))

  4. Solução: Queremos (x - 1)(x + 2) ≥ 0 (positivo ou zero).

    Pela tabela, isso ocorre quando x ≤ -2 ou x ≥ 1.

Conjunto Solução (S): S = {x ∈ ℝ | x ≤ -2 \text{ ou } x ≥ 1}

Dicas Importantes e Pontos de Atenção

  • Símbolos e Intervalos:
    • > ou < : Intervalos abertos ( ) ou bolinha aberta ○ na reta.
    • ≥ ou ≤ : Intervalos fechados [ ] ou bolinha fechada ● na reta.
  • Multiplicação/Divisão por Negativo: Lembre-se SEMPRE de inverter o sinal da desigualdade ao multiplicar ou dividir ambos os membros por um número negativo.
    Ex: -2x < 4 ⇒ x > -2 (dividiu por -2, inverteu < para >)
  • Denominador Diferente de Zero: Em inequações quociente f(x)/g(x), a condição de existência é g(x) ≠ 0. Os valores de x que zeram o denominador nunca farão parte da solução.
  • Estudo do Sinal da Função Quadrática: A análise da parábola (concavidade e raízes) é a chave para resolver inequações do 2º grau. Um esboço rápido sempre ajuda!
  • Organização: Ao resolver sistemas ou inequações produto/quociente, a organização com tabelas de sinais é crucial para evitar erros.
